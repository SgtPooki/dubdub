You get the same API we use

We built every part of Mozu with the same API that we expose to developers, so you have complete access to the operations and entities you need. This API-first development model gives you control over every aspect of the Mozu platform--from themes and applications to custom integrations and admin functionality.

Here’s a sneak peek at our API

  • Rest API access for every entity in the system
  • Application events for every entity change
  • Robust SDKs for every platform
  • Extensible schema for primary system entities
  • Comprehensive theme and widget development platform
  • Extension points to replace system functions with custom implementations
